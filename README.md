# Wignax üêà

**Wignax** is a JAX-based Python library for simulating Open Quantum Many-Body Dynamics using phase-space methods, specifically the Dissipative Truncated Wigner Approximation (DTWA).

Designed to feel familiar to users of QuTiP, Wignax leverages JAX's composable function transformations (`jit`, `vmap`, `grad`) to enable:

* **Massively parallel trajectory simulations** on GPUs/TPUs.
* **Automatic differentiation** through quantum dynamics (useful for quantum control).
* **Symplectic integration** generated automatically from Hamiltonian definitions.
